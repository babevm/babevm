Detailed Description

A thread stack segment. Threads have a list of one or more these.

Field Documentation

◆ body

bvm_cell_t body[1]

Pointer to the first bvm_cell_t of the stack. The actual length of each segment is determined at runtime. The default stack size (in cells, not in bytes) is given in the global bvm_gl_stack_height, which is defaulted from the compile-time define BVM_THREAD_STACK_HEIGHT.

◆ height

size_t height

Height of this stack segment in cells (not bytes).

◆ next

struct _bvmstacksegmentstruct* next

Pointer to the next bvm_stacksegment_t in the list. If NULL, this is the last segment in the list.

◆ top

bvm_cell_t* top

Pointer to the bvm_cell_t past the top of the stack body (as represented by the body field of this structure).
