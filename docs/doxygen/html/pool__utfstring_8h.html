◆ bvm_utfstring_pool_get()

bvm_utfstring_t* bvm_utfstring_pool_get ( bvm_utfstring_t *  str,
bvm_bool_t  add_if_missing 
)

Retrieve a uftstring from the pool. Returns a pointer to the pooled string if it exists. If it does not exist if add_if_missing is BVM_TRUE, the passing in utfstring will be added to the pool and will be the function return value, otherwise (if add_if_missing is BVM_FALSE) NULL will be returned.

Parameters
strthe bvm_utfstring_t to look for
add_if_missing
Returns
the found bvm_utfstring_t if in the pool (or NULL if not), or if not found and add_if_missing is BVM_TRUE, then the same pointer as str

bvm_utfstring_t* bvm_utfstring_pool_get_c ( const char *  data,
bvm_bool_t  add_if_missing

To get a utf string from the pool that matches the given char* (which must be null terminated). If add_if_missing is BVM_TRUE, the char* will be wrapped in a utfstring and added to the pool.

Parameters
datachar * to string - must be null terminated
add_if_missingif BVM_TRUE and the utf string is not in the pool it'll be added.
Returns
a handle to the pooled utf string - will be the same as data if added.
