◆ file_name

bvm_string_obj_t* file_name

The String name of the source file (if any) of the executing method. The compile-time-option BVM_LINE_NUMBERS_ENABLE must be 1 for this to have any other value except NULL. Additionally, the java class file must have been compiled with the java option to include the file name in the class.

◆ line_number

bvm_cell_t line_number

The line number within the source file of the current executing line within the frame segment. The compile-time-option BVM_LINE_NUMBERS_ENABLE must be 1 for this to have any other value except -2 (if the method is native), or -1 (which represents no line number).
