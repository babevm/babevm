Internal command handler logic for ObjectReference/GetValues and ObjectReference/SetValues - both are largely the same.

Note: 'Native' fields (those that hold a pointer on behalf of the VM) are output as INT. When attempting to set the value of a native field, the set operation is ignored.

Parameters
inthe request input stream
outthe response output stream
do_setif BVM_TRUE, the function will set values with an object, if BVM_FALSE function will output values to the output stream.
