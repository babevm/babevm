Push a ptr onto the permanent root stack to stop it from becoming GC'd during a collection. Note there is no 'pop' equivalent - this is performed automatically by the BEGIN/BVM_END_TRANSIENT_BLOCK macros.

ptr- a pointer to the object/data to be pushed.
